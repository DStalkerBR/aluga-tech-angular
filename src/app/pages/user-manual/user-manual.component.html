Manual do Usuário - AlugaTech

Bem-vindo ao AlugaTech! Este manual irá guiá-lo pelas funcionalidades principais da nossa aplicação.

Página Inicial

A página inicial serve como ponto de entrada da aplicação, fornecendo uma visão geral das funcionalidades e oferecendo links rápidos para as principais seções.

Dialog de Login e Registro

Permite que os usuários façam login ou se registrem na plataforma. A página inclui validações para garantir que os dados inseridos são corretos.

Lista de Dispositivos

Exibe uma lista de todos os dispositivos disponíveis para aluguel. Os usuários podem navegar para editar ou deletar dispositivos.

Edição de Dispositivos

Permite que os administradores ou usuários autorizados editem as informações de um dispositivo específico.

Detalhes do Dispositivo

Exibe informações detalhadas de um dispositivo específico, permitindo que os usuários vejam todas as especificações antes de realizar o aluguel.
